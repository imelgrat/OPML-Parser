ParseElementStart()

ParseElementStart(resource  $parser, string  $tagName, array  $attrs)

OPML_Parser::ParseElementStart() The XML tag-open handler. It is used here to parse and store attributes from outline tags

Parameters

resource $parser

A reference to the XML parser calling the handler.
string $tagName

The name of the element (tag) for which this handler is called. If case-folding is in effect for this parser, the element name will be in uppercase letters.
array $attrs

The element's attributes (if any).The keys of this array are the attribute names, the values are the attribute values.Attribute names are case-folded on the same criteria as element names.
